Game Rule

  • This application follows the standard ludo game rule

    • Create Game Room

  • To creat a room just head here and fill the form
  • The creator of the room is player one

    • Join Game Room

  • You can join a room only if you have the room ID
  • To join room head here here and fill the form as appropriate

    • House

  • The house that belongs to a player is indicated with the player name inside the house

    • Turn

  • On start of a new game, any of the player can roll the dice first
  • Afterwards it is controlled by the turn toast
  • The turn toast alerts the player that he can roll the dice after the other player is done counting

    • Roll a Dice

  • To roll a dice click on the dice to roll them.
  • The result are displayed in a small boxes on the top of the game board

    • Counting

  • To count, select the piece you want to count
  • Then select the dice result you want to use to count
  • Then click on count

    • Exit Game

  • Exiting a game implies you intend to exit the game room
  • After clicking on exit, the other player is notified of your exit and the game data is saved

    • Continue Unfinished Game

  • An unfinished game can be continue only if you still have the room ID of the game
  • All you have to do is click on join room and join as player(player one or two) you were before

    • After Wining

  • When a game is won by a player, a modal popup indicating the name of the user that won
  • The player can either exit or play again game
